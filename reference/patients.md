# Randomly Generated Patient Data

A randomly generated data set of "patient" data

## Usage

``` r
patients
```

## Format

### `patients`

A data frame with 150 rows and 4 columns:

- sex:

  Patient sex (M or F)

- stage:

  Stage of disease (I, II, or III)

- site:

  Recruitment site (1 to 10)

- eligible:

  Which treatments are they eligible for? (A, B, C)
