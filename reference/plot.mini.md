# Plots to explore minimisation results

Plots to explore minimisation results

## Usage

``` r
# S3 method for class 'mini'
plot(x, show.plot = TRUE, ...)
```

## Arguments

- x:

  an object of class `mini` as a result of `minimises` with
  `nrow(x) > 0`.

- show.plot:

  logical; if `FALSE` the plot won't be displayed, useful when assigning
  the plot for future use.

- ...:

  other parameters to be passed through to plotting functions.
